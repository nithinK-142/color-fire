Color Fire

  This app allows you to generate and save your favorite colors.
      
-> Generate and save any number of colors.The application displays 
   the generated colors with their corresponding color codes.

-> Copy generated colors to clipboard.

-> Generate random colors and add to favorites, which you like.

-> View favorite colors in a list or grid format

-> Set background color from your favorites.

-> Can also remove them from favorites.

-> Download your favorite colorcodes as json.
